T-MATS: Duct Library Block
This block simulates the performance of a duct using basic fluid dynamic equations and properties.

Duct Inputs:
Duct InputDescription
GasPthCharInGas Path Characteristics Input, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]

Duct Outputs:
Duct OutputDescription
GasPthCharOutGas Path Characteristics Output, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]

Duct Mask Variables:
Duct Mask VariableDescription
dP_MDelta Pressure [frac lost]
