Embedded Target for Freescale 9S12 microcontrollers
    
Timer Unit Block

Block for time-based measurements and signal production

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Timer Unit block gives access to the Enhanced Timer Unit (ETU) of the 9S12. This unit can be operated in two ways:

Used in Input Capture mode, the unit measures the time between any two successive events. At present only rising edges on a selected pin of port PTT qualify as such events. The block output is a measure of the time which has elapsed between the previous two events. A maximum period has to be chosen. Measurements of time intervals longer than this period are invalid. In Input Capture mode, the block icon displays the chosen resolution. This value is determined by dividing the chosen period by 216 = 65536.

In Output Compare mode, the unit is used for signal production. A symmetric square wave signal (50% duty cycle) is produces on the selected pin. The channel number defines the pin of port PTT. As in Input Capture mode, the maximum period divided by 65536 yields the resolution of the square wave signal. In Output Capture mode, the signal level at the block input modulates the signal period. A value of 0 or below switches the pulse train off. A value of 1 and above produces a signal with twice the requested period (on-phase + off-phase). Note that the timer can run at very fast rates. It is possible to produce a square wave signal with a period of only 100 ns! The Sample rate of the block simply defines how often the block input is checked.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

Timer channel
The timer unit connects to the outside world through port PTT. The timer channel parameter defines which pin is used.

Timer mode
Select either Input Capture (IC) mode or Output Compare (OC). Input Capture is used to measure durations inbetween successive events (rising edges on the associated input pin). Output Compare is used to produce symmetric square wave signals with (twice) the requested period (on-phase + off-phase -> 2).

Timer base period (seconds)
In Input Capture mode, the base period defines the maximum allowed duration between successive rising edges. Events which lie outside this window produce incorrect measurements. In Output Compare mode, the base period defines half the period of the generated square wave signal.
