T-MATS: Normaized Error Library Block

The Normaized Error block determines a difference and then normalizes to the first input. This is useful for quickly creating error terms to be used with the T-MATS solvers.

$$\frac{(X_1-X_2)}{X_1}= NError$$

NEr Inputs:
NEr InputsDescription
X1Input 1
X2Input 2

NEr Outputs:
NEr OutputDescription
NErNormalized Error
