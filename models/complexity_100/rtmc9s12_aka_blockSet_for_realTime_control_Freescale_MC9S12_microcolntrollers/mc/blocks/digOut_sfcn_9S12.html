Embedded Target for Freescale 9S12 microcontrollers
    
Digital Output

Driver block for general purpose digital outputs (GPiO)

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Digital Output block controls the logic level between 1 and 8 pins of the chosen general purpose digital I/O port of the 9S12. The number of block inputs changes with the chosen number of digital output pins.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

Port
Select one of 8 available ports (on the 9S12DP256): PORTA, PORTB, PTH, PTJ, PTM, PTP, PTS and PTT.

Pin
Specify the desired output pins (0-7). Multiple pins can be specified as a vector. The block initialization code ensures that only actually available pins can be chosen.

On-Threshold
Block input signals above this threshold are interpreted as logic high. Choosing a value Voff < Von causes the output characteristics of the selected port to exhibit hysteresis.

Off-Threshold
Block input signals below this threshold are interpreted as logic low. Choosing a value Voff < Von causes the output characteristics of the selected port to exhibit hysteresis.
