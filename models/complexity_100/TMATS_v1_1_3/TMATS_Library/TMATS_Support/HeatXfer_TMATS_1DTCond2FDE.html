T-MATS: 1-D Trans Cond Model 2mat(FDE) Block
This function models the transient conduction of a plate ,consisting of two different materials, A and B coinciding with inner and outer respectively, with fluid on either side via the finite difference method (explicit).
Export values are calculated based on a single time step. If additional dynamic behavior is required a simulink memory block may be used to update the temperature initial conditions.
Units must be specified consistantly. (i.e. If temperature is in Celsius all temperature units must in Celsius)
1-D Trans Conduction 2mat Inputs:
1-D Trans Conduction 2mat InputsDescription
TinnerInner Fluid Temperature
TouterOuter Fluid Temperature
TiVecInitial Temperature at each Node ((a+b)x1)

1-D Trans Conduction 2mat Outputs:
1-D Trans Conduction 2mat OutputDescription
TAavgFinal Average Temperature within the Plate A
TBavgFinal Average Temperature within the Plate B
TfVecFinal Temperature at each Node ((a+b)x1)
XdistPlacement of each Node in A (ax1)
XdistPlacement of each Node in B (bx1)
