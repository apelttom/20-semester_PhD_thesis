Embedded Target for Freescale 9S12 microcontrollers      
Analog to Digital Converter, ADC

Input driver for Analog-to-Digital Converter (ADC) unit

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Analog to Digital Converter (ADC) block reads up to 8 channels of data from one of two 8-channel ADC banks. The resolution can be chosen to be either 8 bits or 10 bits. The block output can be configured for raw data (8-bit or 16-bit), normalized to the interval [0 ... 1] or normalized to the interval [0 ... 5]. The number of block outputs changes with the selected number of channels.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

ADC bank
Select either bank 0 or 1. Each bank has 8 channels.

Channels
Specify input channel(s) 0-7. Specify multiple channels as a vector or using a hyphen.

ADC resolution
Select a resolution of either 8 bits or 10 bits. The default is an 8-bit resolution.

Normalization
The block can be set up to emit raw data (8-bit or 16-bit, this depends on the chosen ADC resolution), values in the range from '0 to 1' or values ranging from '0 to 5'.
