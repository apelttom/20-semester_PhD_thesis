Embedded Target for Freescale 9S12 microcontrollers
    
Pulse Width Modulated Output, PWM

Block to produce Pulse-Width Modulated (PWM) output signals

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Pulse-Width Modulated (PWM) output block can be configured to generate PWM signals, either with an 8-bit or with a 16-bit resolution. Up to 8 PWM channels are available in 8-bit mode, 4 channels in 16-bit mode. The block accepts signed input signals. The magnitude of these signals defines the duty cycle of the PWM signal. The sign information is displayed on one of the digital output pins.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

Resolution
The resolution of the chosen PWM channel. There are 8 channels when the resolution is 8 bits; this reduces to 4 channels in 16-bit mode.

PWM period
Specifies the period of the PWM signal. Period values above 0.699 seconds require the use of a 16-bit PWM; below this threshold, both 8-bit as well as 16-bit operation is possible. In 16-bit mode, the maximum admissible period is 179 seconds.

Pulse pin (PTP)
The PWM unit uses port P (PTP)  to emit the pulse train. As each 16-bit channel ties up 2 PWM timers, only every second pin of PTP can be chosen in 16-bit mode ('1', '3', '5' or '7'). This limitation does not apply to the 8-bit mode.

Sign port
The PWM block accepts signed block input signals. The magnitude of the input signal defines the duty cycle of the PWM output. The sign information is displayed on a chosen pin of the Sign port.

Sign pin
The pin within the associated Sign port. Sign information is displayed on this pin: Negative block input signals drive this pin high, positive block input signals set this pin to low.

Saturation input level (Simulated voltage)
Block input signals with a magnitude above this level are mapped to a duty cycle of 100%.

Signal monitoring
Ticking this checkbox causes the block to display an optional output signal which can be used to monitor the PWM signal on a Simulink Scope block (External Mode). This feature is only useful when the block Sample time is reasonably short with respect to the PWM period.
