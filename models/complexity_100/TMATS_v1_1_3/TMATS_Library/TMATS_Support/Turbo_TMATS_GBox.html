T-MATS: Gear Box Library Block
This block simulates the performance of a gear box using basic gear properties.
Gear Ratio = NmechA/NmechB = TrqB/TrqA

Gear box Inputs:
Gear box InputDescription
TorqATorque at gearbox side A [lbf*ft]
NmechBMechanical shaft speed at gearbox side B [rpm]

Gear box Outputs:
Gear box OutputDescription
NmechAMechanical shaft speed at gearbox side A [rpm]
TorqBTorque at gearbox side B [lbf*ft]

Gear box Mask Variables:
Gear box Mask VariableDescription
R_MGear Ratio [frac]
Eff_MGear box efficiency from A to B [frac]
