T-MATS: Limtit Controller (PI)
This block contains a PI controller with anti-windup protection. Gains, Kp and Ki, are defined as gains inside the block.

LC Inputs:
LC InputDescription
Limit Value Limit value
Input_sensed Sensed input (to be limited via the effector)
Eff_dmd Effector actual demand

LC Outputs:
LC OutputDescription
Eff_Limit Effector Limit value
Active Limiter is: 1 - active, 0 - not active

LC Mask Variables:
LC Mask VariableDescription
Kp_MProportional gain
Ki_MIntegral gain
PI_IC_MInitial effector value
WGain_MIntegrator windup gain
Ts_MSampling time (sec)
