Embedded Target for Freescale 9S12 microcontrollers
    
On-board Digital to Analog Converter, Dragon-12 (Rev-E)

Driver for the on-board Digital to Analog Converters (DAC) of Dragon12 (Rev-E) boards. These DACs are connected to the SPI bus of the MC9S12.

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The On-board Digital to Analog Converter block gives access to the two on-board D/A Converters on the Dragon-12 (Rev-E) boards.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

DAC channel
The Dragon-12 (Rev-E) features two on-board D/A Converters, DAC0 and DAC1.

Saturation level (block input)
This value represents the model sided full-scale value of the on-board DAC. Block input signals above this threshold will yield a full-scale output of 5 V. In other words, the Saturation level defines the model sided output range (0 - Saturation level).
