Embedded Target for Freescale 9S12 microcontrollers
    
Digital Input

Driver block for general purpose digital inputs (GPIo)

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Digital Input block reads up to 8 channels of data from the chosen general purpose digital I/O ports of the 9S12. The number of block outputs changes with the chosen number of digital input pins.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

Port
Select one of 8 available ports (on the 9S12DP256): PORTA, PORTB, PTH, PTJ, PTM, PTP, PTS and PTT.

Pin
Specify the desired input pins (0-7). Multiple pins can be specified as a vector. The block initialization code ensures that only actually available pins can be chosen.
