T-MATS: 0-D Trans Conduction model (Lumped Cap) Block
This function models the transient conduction of a lumped system via the lumped capacitance method.
Export values are calculated based on a single time step. If additional dynamic behavior is required a simulink memory block may be used to update the temperature initial conditions.
Units must be specified consistantly. (i.e. If temperature is in Celsius all temperature units must in Celsius)
0-D Trans Conduction Inputs:
0-D Trans Conduction InputsDescription
TinfOut of Lumped System Temperature
TiInitial Lumped System Temperature

0-D Trans Conduction Outputs:
Heat Transfer Lumped System OutputDescription
TfFinal Lumped System Temperature
