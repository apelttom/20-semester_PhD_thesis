Servo Motor Driver Block

Driver block for servo motors

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Servo Motor PWM Output block can be configured to produce Pulse-Width Modulated signals with an 8-bit or 16-bit resolution. Up to 8 PWM channels are available (8-bit mode, 4 channels in 16-bit mode). The block accepts unipolar input signals. The magnitude of these signals defines the duty cycle of the PWM signal, ranging from the specified Minimum Pulse Width (0) to the specified Maximum Pulse Width (saturation input level). The specified resolution (8 bit / 16 bit) applies to the entire period of the PWM signal, i. e. the smallest increment is PWMperiod / 2Resolution.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

PWM period
Period of the Pulse-Width Modulated signal. This is typically a value of 20 ms or 40 ms.

Minimum Pulse Width
This is the pulse width of the shortest pulses, corresponding to the minimum position of the servo motor.

Maximum Pulse Width
This is the pulse width of the longest pulses, corresponding to the maximum position of the servo motor.

Resolution
The resolution of the chosen PWM channel. There are 8 channels at 8-bit or 4 channels at 16-bit.

Pulse pin
The PWM unit uses port P (PTP)  to display the pulse sequences. As each 16-bit channel ties up 2 PWM timers, only every second pin of PTP can be chosen in 16-bit mode ('1', '3', '5' or '7'). This constraint does not apply to the 8-bit mode.

Saturation input level (Simulated voltage)
Block input signals with a magnitude above this level are mapped to a duty cycle of 100%.
