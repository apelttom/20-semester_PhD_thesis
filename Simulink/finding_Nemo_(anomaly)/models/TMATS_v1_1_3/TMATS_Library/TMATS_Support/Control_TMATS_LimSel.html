T-MATS: Limit Selection
This block uses limit inputs to adjust a requested effector value.

LS Inputs:
LS InputDescription
Request Value Requested Value
Limit Values Values of limiters as a vector. These values may be changed for different conditions (1xn)

LS Outputs:
LS OutputDescription
Demand Value Final value after limits are applied
ActiveLimit Value of the active limit. The limit number will coincide with it's location in the Limit Values input vector

LS Mask Variables:
LS Mask VariableDescription
SelectVec_MValues of 1 denote a topping limit and -1 denote a bottoming limit. The input must be in the form of a vector with the same number of values as the Limit Values input vector. (1xn)
LimitsEn_M Enable limits: 1- limits active, 0- limits disabled (1xn)
