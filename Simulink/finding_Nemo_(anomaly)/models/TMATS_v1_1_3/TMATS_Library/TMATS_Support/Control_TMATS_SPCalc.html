T-MATS: Set Point Calculation (Tbl)
This block contains a set point table look up based on an external request and a manual override.

LS Inputs:
LS InputDescription
External Request External system request for table lookup
Determiner Signal Determiner signal for table lookup
Manual_En Enable manual input: 1- manual input active, 0- use value based on exernal request
Manual Setpoint Manual Setpoint: Setpoint used when manual mode is enabled

LS Outputs:
LS OutputDescription
Cntrl Setpoint Setpoint for use by controller

LS Mask Variables:
LS Mask VariableDescription
ERVec_MExternal request vector (for table lookup)(1xm)
DSVec_MDetermine signal vector (for table lookup)(1xn)
SPArray_MSetpoint array (mxn)
