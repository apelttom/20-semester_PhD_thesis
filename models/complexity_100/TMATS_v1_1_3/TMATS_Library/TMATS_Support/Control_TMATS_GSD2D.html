T-MATS: Gain Scheduler double 2-D Library Block
This block a double 2 dimensional gain scheduler. Input are two critera and a common critera that are used for two 2-D lookups. Output contains the result of the table lookups. In general these two results will be multiplied together and can be entered into either the Ki or Kp input of the T-MATS setpoint controller block. Additionally the outputs could be separated and used as inputs to both Kp and Ki.

GS Inputs:
GS InputDescription
CriteraK1 Critera for gain K1
CriteraK2 Critera for gain K2
CriteraCommon Critera common for K1 and K2

GS Outputs:
GS OutputDescription
K Gain K from lookups, size 2 vector [K1 K2]

GS Mask Variables:
GS Mask VariableDescription
CCVec_MCommon critera vector (1xn)
CK1Vec_MGain 1 critera vector(1xm)
K1Array_MGain 1 array (mxn)
CK2Vec_MGain 2 critera vector(1xp)
K2Array_MGain 2 array (pxn)
