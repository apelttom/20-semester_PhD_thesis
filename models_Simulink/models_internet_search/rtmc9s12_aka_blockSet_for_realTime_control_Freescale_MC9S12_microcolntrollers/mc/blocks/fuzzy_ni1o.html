Embedded Target for Freescale 9S12 microcontrollers
    
Fuzzy Control Block

A simple Fuzzy Inference System evaluation block

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


Block to combine n input signals using a Mamdani Fuzzy Inference System (FIS) to produce 1 output signal (-> ni1o, n is currently limited to 3). The generated code is very fast as it makes use of the special FIS capabilities of the 9S12 (at assembler level). It is therefore possible to experiment with simple fuzzy control systems while not having to purchase The MathWorks Fuzzy Control toolbox. Definition of membership functions and the rule table follows the style of FlouLib, a freeware fuzzy toolbox which is available on the net.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

Number of inputs
Up to 3 input signals can currently be processed.

MSF input 1 (filename)
Name of a text file which defines the MemberShip Functions of input 1. The input MSFs are used during fuzzification. See the FlouLib manual or the manual of this toolbox (rtmc9s12-target) for how these MFS are to be defined.

MSF input 2 (filename)
Name of a text file which defines the MemberShip Functions of input 2. The input MSFs are used during fuzzification. See the FlouLib manual or the manual of this toolbox (rtmc9s12-target) for how these MFS are to be defined.

MSF input 3 (filename)
Name of a text file which defines the MemberShip Functions of input 3. The input MSFs are used during fuzzification. See the FlouLib manual or the manual of this toolbox (rtmc9s12-target) for how these MFS are to be defined.

MSF output (filename)
Name of a text file which defines the MemberShip Functions of the output. The output MSFs are used during de-fuzzification. See the FlouLib manual or the manual of this toolbox (rtmc9s12-target) for how these MFS are to be defined.

Rule base (filename)
Name of a text file which describes the rule base used during fuzzy inference. See the FlouLib manual or the manual of this toolbox (rtmc9s12-target) for how to define a valid rule base.
