T-MATS: Simple PI controller Library Block

PI Inputs:
PI InputDescription
Input_demand R - input demanded value
Input_sensed Y - input of plant output to be controlled

PI Outputs:
PI OutputDescription
Effector Demand U - output to plant Effector

PI Mask VariableDescription
Kp_MProportional gain
Ki_MIntegrator gain
