T-MATS: Splitter Library Block
This block simulates the performance of a splitter/branch using basic flow fluid dynamic equations.

Splitter Inputs:
Splitter InputDescription
MFPGasPthCharInGas Path Characteristics of the Main Flow Path, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
BPRBranch Pressure Ratio, 0 - no branch flow
Tyically this input is a system independent and is derived from an iterative solver.

Splitter Outputs:
Splitter OutputDescription
BranchGasPthCharOutGas Path Characteristics of the Branch, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
MFPGasPthCharOutGas Path Characteristics of the Main Flow Path, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
