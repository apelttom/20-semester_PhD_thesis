T-MATS: Shaft Library Block
This block simulates the performance of a shaft using basic dynamic equations and properties.

Shaft Inputs:
Shaft InputDescription
Torqueinput torque from torque source or sink [lbf*f]
NmechInMechanical shaft speed[rpm].
This input is simply passed through to the output and is needed only for
understandability purposes. Calculation of Nmech is completed with the
integration of shaft output Ndot.

Shaft Outputs:
Shaft OutputDescription
NmechOutMechanical shaft speed[rpm], equal to the input shaft speed.
Ndotmechanical shaft speed acceleratoin [rpm/sec]

Shaft Mask Variables:
Turbine Mask VariableDescription
Shaft_Inertia_M Shaft Inertia Term [slugs*ft^2]
