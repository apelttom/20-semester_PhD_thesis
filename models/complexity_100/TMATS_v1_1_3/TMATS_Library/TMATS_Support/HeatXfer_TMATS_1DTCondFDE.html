T-MATS: 1-D Trans Cond Model(FDE) Block
This function models the transient conduction of a plate with fluid on either side via the finite difference method (explicit).
Export values are calculated based on a single time step. If additional dynamic behavior is required a simulink memory block may be used to update the temperature initial conditions.
Units must be specified consistantly. (i.e. If temperature is in Celsius all temperature units must in Celsius)
1-D Trans Conduction Inputs:
1-D Trans Conduction InputsDescription
TinnerInner Fluid Temperature
TouterOuter Fluid Temperature
TiVecInitial Temperature at each Node (mx1)

1-D Trans Conduction Outputs:
1-D Trans Conduction OutputDescription
TavgFinal Average Temperature within the Plate
TfVecFinal Temperature at each Node (mx1)
XdistPlacement of each Node (mx1)
