T-MATS: 1st Order Actuator Library Block

This block models a generic actuator with a first order lag.

$$ \frac {Y(s)}{X(s)} = \frac {Gain}{\tau s+1} $$

Additionally nonlinearities have been included to allow for actuator deadbands, rate limits and saturation points. Deadbands are used to simulate the mechanical play in the system. Rate limits can be set to simulate physical movement limitations of the actuator.

1-O Actuator InputDescription

1-O Actuator Outputs:
