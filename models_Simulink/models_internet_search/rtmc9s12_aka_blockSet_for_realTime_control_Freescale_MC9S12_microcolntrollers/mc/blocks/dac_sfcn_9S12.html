Embedded Target for Freescale 9S12 microcontrollers
    
Digital to Analog Converter, DAC

Driver for Digital to Analog Converters (DAC), connected to the Inter-IC (I2C) bus.

Library

Real-Time toolbox for 'Freescale mc9s12' based targets (rtmc9s12-Target)

Description


The Digital to Analog Converter block gives access to the two serial D/A Converters (Analog Devices AD5311) on the protective circuit board which is used with the Dragon-12 boards at the University of Adelaide, School of Mechanical Engineering.

Dialog Box


Sample time
Sample time of the block. The Sample time (in seconds) defines the rate at which the block is visited by the real-time scheduler of the target. The value of the Sample time has to be an integer multiple of the base sample time of the model. The shortest achievable sample time can be determined using the display timing signals option. This option is described in the manual of the toolbox in section 2.3 Code generation options.

DAC channel
Currently the protective circuit board of the Dragon-12 features two D/A Converters, DAC0 and DAC1. This board has been developed at the School of Mechanical Engineering of the University of Adelaide. Please see the download section of our robotics web page for further details (schematics, parts lists, etc.).

Saturation level (block input)
This value represents the model sided full-scale value of the DAC. Block input signals above this threshold will yield a full-scale output of 5 V. In other words, the Saturation level defines the model sided output range (0 - Saturation level).
