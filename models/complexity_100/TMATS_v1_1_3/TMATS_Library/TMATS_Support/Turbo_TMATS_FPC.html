T-MATS: Flow Path Characteristics Library Block
This block creates a flow path characteristics bus for input into the system.

Flow Path Inputs:
Flow Path InputDescription
Wgas path flow[pps]
TtTotal Temperature [degR]
PtTotal Pressure [psia]
FARCombusted Fuel to Air Ratio [frac]

Flow Path Outputs:
Flow Path OutputDescription
GasPthCharOutGas Path Characteristics Output, 5x1 bus/vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
