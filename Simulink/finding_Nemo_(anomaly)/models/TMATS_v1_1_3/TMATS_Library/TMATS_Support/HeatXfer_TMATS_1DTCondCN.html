T-MATS: 1-D Trans Cond Model(FDI) Block

This function models the transient conduction of a plate with fluid or additional plates on either side via the finite difference method (Crank-Nicolson).

Setup requires the use of an external solver for determining Next temperature states.

This block is setup to be a general use material that is devided into 3 regions, A,B,C.

Region A can be defined as a fluid or a solid material at the inner boundary. If it is a fluid, a TinnerNext is set to a constant. If it is defined as a solid the thermodynamic properties need to be defined and TinnerNext needs to be connected to the adjacent node of another 1-D Trans Conduction Model (CN) block.
Region B is the material as defined in the General Tab
Region C is defined similarly to A, however it is considered the outer material.

Export values are calculated based on a single time step. If additional dynamic behavior is required a simulink memory block may be used to update the temperature initial conditions. Units must be specified consistantly. (i.e. If temperature is in Celsius all temperature units must in Celsius)
